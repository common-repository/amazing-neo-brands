Amazing Neo is one of the best icon font by Amazing Team. This plugin allows you to insert brands/social icons in any widget area. : icon, font, amazing neo, amazing font, icon font.


== Description ==

Amazing Neo Brands Social Icons is an easy to use, display brands/social icons that link visitors to your social media profiles. easily customize the color and size of your icons, as well as align them to the left, center, or right, all from the widget area.
for more details [click the link](https://amazingneo.com "Amazing Neo")


== Installation ==
Install like any other plugin, directly from your plugins page.

1. Upload the plugin files to the `/wp-content/plugins/amazing-neo-brands` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. In your Widgets menu, Drag the widget labeled "Amazing Neo /Brands Social Icons" into a widget area.
1. Configure the widget by choosing a title, icon size and color, and the URLs to your social media profiles.
